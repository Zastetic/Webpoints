INSTRÇÕES PARA CRIAÇÃO DE ARQUIVOS NO PROJETO

templates  --> arquivos .html
static/js  --> arquivos .js
static/css --> arquivos .css
app.py     --> servidor

Para iniciar o server:
 1. Inicie o terminal no vscode na pasta do projeto (ctrl + shift + "")
 2. baixe o flask e sqlalchemy (pip install flask / pip install sqlalchemy)
 3. Rode o comando: python app.py
 4. Entre no endereço fornecido. (algo como "http://127.0.0.1:5000"). Informações apareceram no terminal. Leia-as.
